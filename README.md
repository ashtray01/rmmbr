показыает определения чтобы запомнить
<br>

умеет вытаскивать и анализировать определения<br>
из .docx, .xlsx, .pdf, .txt<br>
руководствуется правилами regex:<br>
<br>^([^—–-]+)\s*[-—–]\s*(.+)$
<br>^([^:]+)\s*:\s*(.+)$
<br>^([^—–:]+)\s*(?:—|–|-)?\s*это\s*(.+)$
<br>^([^,]+),\s*то есть\s*(.+)$
<br>^([^\(]+)\s*\(([^)]+)\)$
<br>^([^—–:]+)\s*(?:—|–|-)\s*(.+)$
<br>
